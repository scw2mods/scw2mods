# Icons Directory

Ce dossier contient les icônes du site.

## À Faire

### Favicon
Créer un favicon pour le site :
- **Nom** : `favicon.png`
- **Taille** : 32x32px ou 64x64px
- **Format** : PNG avec transparence
- **Design** : Logo ou icône représentant SCW2

Tu peux utiliser des outils en ligne comme :
- https://favicon.io/
- https://realfavicongenerator.net/

### Icônes de catégories (optionnel)
Les catégories utilisent actuellement des emojis, mais tu peux ajouter des icônes personnalisées si besoin.
